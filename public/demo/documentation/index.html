Doccure provides online appointment scheduling system for your patients. It also called Multiple Clinic Booking for doctor appointment which has many features to be provided for patients who are seeking particular doctor or specialist. This template consists of the details around the city so user can find the nearest clinic and doctor. It is classified the details as per the categories of the clinic services. The information is like details of the doctors, clinics, images and so on. User has option to give ratings and comments about the clinic.

HTML Installation

  • After unzipping Template Files, you will find our product Doccure->HTML(Source code), Documentation, and License in the following Main Files folder.
  • Open HTML folder, copy all files and paste in your localhost or server root folder. Now you can run this Template in any web browser from your PC or Laptop.

For example:
localhost/doccure/index.html or www.example.com/index.html

Requirements

Working with our Doccure product, before going to edit all the HTML, CSS and JS files you will need a text editor.

In this case, we recommended the Sublime Text 3 editor or Notepad++ as it allows you to edit the files efficiently and all the editable files are located in "Doccure->HTML" folder.

File Configuration

How to change Logo ?

You can change the logo or replace it from img folder or you can edit it from HTML file.

<div class="navbar-header">
  <a href="index.html" class="navbar-brand logo">
	<img src="assets/img/logo.png" class="img-fluid" alt="Logo">
  </a>
</div>

How to change Header background color ?

You can change the header background color in style.css as per your wish

How to change Google Font ?

Go to style.css and line number 55. Here we have the default font family Mada

@import url('https://fonts.googleapis.com/css?family=Mada:400,500,600,700&display=swap');
